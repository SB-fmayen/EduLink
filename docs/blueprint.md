# **App Name**: EduLink

## Core Features:

- User Authentication: Secure login and authentication using Firebase Authentication with email/password and role-based access control (Admin, Teacher, Parent, Student).
- School Management: Admin module to create and manage schools, configure grades, sections, and subjects.
- Academic Management: Register students and teachers, assign courses and schedules, and manage attendance (manual or QR code).
- Grading System: Teachers can input grades, generate report cards automatically, and view statistics per course and student.
- Communication Portal: Internal chat system for parents and teachers, push notifications for announcements, and circular/event publishing using Firebase Cloud Messaging.
- Financial Management: Control tuition fees, online payments via PayPal/cards, and generate financial reports.
- Educational Platform: Upload assignments and digital resources, conduct online exams, and track academic progress.

## Style Guidelines:

- Primary color: Light Gray (#D3D3D3) for a clean, neutral interface. It provides a subtle base that's easy on the eyes and ensures readability.
- Background color: White (#FFFFFF) for a light mode default.
- Accent color: Dark Blue (#00008B) to highlight interactive elements and important information.
- Body and headline font: 'Inter', a sans-serif, providing a modern and clean aesthetic.
- Simple, clear icons to represent different functions and modules.
- Clean, modular layout for easy navigation and scalability.
- Subtle transitions and animations to enhance user experience without being distracting.
